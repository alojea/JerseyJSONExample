Example Webservice using Jersey-Jackson

Producer 
Just enter this url in the browser
http://localhost:8080/JerseyJSONExample/rest/jsonServices/print/carlos

Consumer 
In postman using method post send a JSON post message to this URL
http://localhost:8080/JerseyJSONExample/rest/jsonServices/send

Body/RAW/JSON application

{"id":1,"firstName":"carlos","age":22,"lastName":"Diaz"}
